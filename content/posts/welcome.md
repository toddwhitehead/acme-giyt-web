We're excited to announce the launch of our new static website built with Hugo!

## Why Hugo?

Hugo is incredibly fast, flexible, and perfect for building modern static websites. Here are some benefits:

- **Lightning Fast**: Hugo builds sites in milliseconds
- **Flexible**: Easy to customize and extend
- **Secure**: No database means no database vulnerabilities
- **Simple Deployment**: Deploy anywhere - GitHub Pages, Netlify, Vercel, or any static host
